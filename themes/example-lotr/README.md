# example-lotr

Placeholder for a Lord of the Rings-themed concept pack.

A LOTR concept pack might map:
- `profiles` → "Realms"
- `sessions` → "Journeys"
- `tools` → "Artifacts"
- `kanban` → "Quest Board"
